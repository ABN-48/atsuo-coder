# Atsuo Coder

## Environment

- Ubuntu 20.04 or 22.04
- Node.js (v18~)
- Docker
- MySQL 8.0
- Redis

## Install

### AtsuoCoder

```sh
git clone https://github.com/atsuo-tech/atsuo-coder
cd atsuo-coder/api
npm run build
```

### MySQL

```sh
npm run install:mysql
```

### Redis

```sh
npm run install:redis
```

## Initialize

### MySQL

```sh
npm run init:mysql
```

## Run

```sh
npm run start
```
